# Task Tracker - Simple Checklist App
Manage your tasks in one web-app, mark them complete with just one click!

## Features
<ul>
  <li>Easily jot down the title and description of a task</li>
  <li>Set the deadline of each task set</li>
  <li>Tasks added are displayed in a clean table that adjusts automatically as more tasks are added</li>
</ul>
